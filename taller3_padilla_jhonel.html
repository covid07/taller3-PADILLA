Introducción

Este documento presenta un análisis exploratorio de un conjunto de datos sobre herpetofauna del Caribe colombiano recolectados durante el año 2023. El dataset incluye registros de anfibios y reptiles provenientes de diferentes ecosistemas de los departamentos del Atlántico y Magdalena.

En este análisis se estudiarán variables como ecosistema, clase, peso corporal y longitud hocico-cloaca. Además, el archivo utilizado fue cargado mediante el comando read.csv() en R.

Gráfico 1 — Registros por ecosistema

barplot(
  table(herp$ecosistema),
  col = "darkgreen",
  main = "Registros por ecosistema",
  xlab = "Ecosistema",
  ylab = "Frecuencia"
)

El ecosistema con mayor cantidad de registros presenta una mayor representación dentro del conjunto de datos analizado.

Gráfico 2 — Peso corporal por clase

boxplot(
  peso_g ~ clase,
  data = herp,
  col = c("orange", "skyblue"),
  main = "Peso corporal por clase",
  xlab = "Clase",
  ylab = "Peso (g)"
)

El gráfico permite comparar la distribución del peso corporal entre anfibios y reptiles.
